# Employee Management System

## Overview

The Employee Management System is a PostgreSQL-based database project that manages employees, departments, payroll, attendance, performance reviews, and promotions.

This project demonstrates real-world HRMS (Human Resource Management System) concepts while showcasing advanced SQL skills including:

* Database Design
* Constraints
* Joins
* Subqueries
* Views
* CTEs
* Window Functions
* Analytics Reporting

---

## Features

### Employee Management

* Employee Profiles
* Department Assignment
* Position Management
* Manager Hierarchy

### Attendance Management

* Daily Attendance Tracking
* Attendance Reports

### Payroll Management

* Salary Records
* Bonuses
* Deductions
* Net Salary Calculation

### Performance Management

* Employee Reviews
* Performance Scores
* Promotion Eligibility

### Analytics Dashboard

* Department Analytics
* Payroll Analytics
* Performance Analytics
* Executive KPIs
