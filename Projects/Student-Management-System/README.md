# 🎓 Student Management System

A comprehensive PostgreSQL database project designed to manage students, courses, enrollments, attendance, examinations, and performance analytics.

## Features

* Student Management
* Course Management
* Instructor Management
* Enrollment Tracking
* Attendance Monitoring
* Examination Management
* Marks Management
* Analytics Dashboard
* Student Performance Reports

## Technologies Used

* PostgreSQL
* SQL
* pgAdmin

## Database Modules

### Students

Stores student information.

### Instructors

Stores faculty information.

### Courses

Stores course information.

### Enrollments

Maps students to courses.

### Attendance

Tracks student attendance.

### Exams

Stores examination information.

### Marks

Stores examination results.
